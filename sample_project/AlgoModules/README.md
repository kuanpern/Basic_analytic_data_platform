## AlgoModule
This directory deposits algorithm, statistical module and predictive models for the project.

### File tree
```
/
├── BI_analyses                      : parent folder for business intelligence analytics
│   └── templates                    : templates for business intelligence analytics
├── predictive_analytics             : parent folder for predictive analytics
│   ├── templates                    : templates for predictive analytics
│   │   ├── data_selection           : template for data selection
│   │   ├── feature_selection        : template for feature selection
│   │   ├── train_test_validate      : template for train-test-validate workflow
│   │   └── error_analysis           : template for error analysis
│   └── weather_predictor            : simple demo of HMM model
└── statistical_analyses             : parent folder for statistical analytics
    └── templates                    : templates for statistical analytics
        └── survival_analysis.ipynb  : simple example of a survival analysis model
```
